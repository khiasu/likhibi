# Likhibi Keyboard

[![Download APK](https://img.shields.io/badge/Download-APK-green?style=for-the-badge&logo=android)](https://github.com/khiasu/likhibi/raw/main/apk/likhibi-keyboard.apk)

Likhibi is an Android custom keyboard built specifically for the Nagamese language. It prioritizes a fast, highly responsive typing experience coupled with a modern, dynamic aesthetic.

## Overview
This repository contains the initial release of the Likhibi Keyboard. The current version establishes the core infrastructure, including:
- A dynamic, programmatic theming engine.
- An offline, low-latency predictive text system.
- System-wide haptic and distinct auditory feedback.
- A clean, edge-to-edge native settings interface.

## Roadmap
This is an early-stage build. Future development will focus heavily on linguistics:
- In-depth Nagamese text meaning research.
- Advanced linguistic analysis to improve prediction accuracy and context awareness.
- Further refinements to the offline machine learning model.

## Installation
You can build the project directly using Android Studio and Gradle, or install the compiled APK directly onto your Android device.
